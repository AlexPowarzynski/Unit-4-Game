Do you remember?

The goal of this game is to match your "Total Meme Points" to the "Goal Meme Points". The tricky part of this game is that each meme down below has a random point value between 1-12. This is random and resets each round. Make sure you don't go over the "Goal Meme Points" or you'll get a loss. So, do you remember each value of the buttons?

  • Match your points to the goal points to win that round.
  • Don't go over the goal points, otherwise you get a loss.
  • The buttons are a random value between 1-12 and randomize each round.
  • The reset game button will reset the wins/losses as well as randomize everything each time you click it.
  • Get 10 Wins/Losses and the game will reset.
